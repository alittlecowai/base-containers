# Base Containers
This repository serves as a base for custom-made container images used in AI-powered applications.

[In Progress]: A CICD pipeline to build the various images and push into Github container registry.

## Containers List

### Backend AI Python [In Progress]
For AI applications that expose a backend API via REST or websockets

### Full Stack AI Python
For AI applications that use SQL database as well as Python to serve frontend webpages and backend APIs

### Full Stack AI Python (NoSQL) [In Progress]
For AI applications that use NoSQL database as well as Python to serve frontend webpages and backend APIs

### Postgres
Containers for Postgres SQL database
